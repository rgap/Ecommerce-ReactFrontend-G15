# ECOMMERCE-G15

An e-commerce website developed as a final project for a Fullstack bootcamp.

This only covers the Front-End side so far.

## Online Demo

https://ecommerce-react-g15.vercel.app/

## Tools & Frameworks

- React JS
- Tailwind 3.3
- Formik
- Toastify
- Figma
- HTML5/CSS3
